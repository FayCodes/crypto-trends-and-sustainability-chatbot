# Crypto Trends and Sustainability Chatbot

A rule-based chatbot that provides insights into cryptocurrency market trends and sustainability factors to support informed decision-making.

---

## Overview

This chatbot analyzes predefined cryptocurrency data using structured rule-based logic to generate insights on trends and sustainability.

It demonstrates foundational AI concepts through decision-based programming without using machine learning models.

---

## Features

- Modular Python structure (data.py, bot.py, main.py)
- Predefined cryptocurrency dataset for analysis
- Rule-based decision engine for generating insights
- Interactive terminal chatbot interface
- Lightweight and runs on VS Code, Google Colab, and GitHub

---

## Project Structure

- data.py — Cryptocurrency dataset  
- bot.py — Chatbot logic  
- main.py — Entry point  
- images/ — Chatbot screenshots  
- README.md — Documentation
